### About

I'm a career-changer moving from ~9 years in web, design and client delivery into **network and security operations**. I learn by doing, so instead of only reading I built a physical home lab on real enterprise gear and run, break and rebuild it every day.

The **networking runs my home for real**. The **SOC stack is a deliberate learning environment** I stood up to study blue-team fundamentals — I've built the infrastructure and I'm actively learning to operate it well. This GitHub is where I document that, sanitized.

- 🎓 **CCNA Certified (200-301)** — Cisco
